Single Page Application (SPA)의 MathJax 적용

작성자: 여행 초짜
작성일: 2023.02.05

이번 글에서는 single page application (SPA) 웹 페이지에서 MathJax를 사용하는 방법에 대해 알아보겠습니다. 먼저 MathJax는 HTML에서 수식을 사용할 수 있게 도와주는 JavaScript의 라이브러리입니다. 기본적으로 MathJax는 LaTeX 문법과 아주 흡사합니다. 그리고 직접 구성한 웹페이지에서 수식을 사용하기 위해서는 MathJax가 필수입니다.

먼저 HTML에 MathJax를 사용하기 위해서는 <body> tag 안에 아래 코드를 입력해주면 됩니다. 

<body>
    <script src="//cdn.mathjax.org/mathjax/latest/MathJax.js?config=TeX-AMS-MML_HTMLorMML"></script>
</body>


다른 페이지로 넘어갈 때마다 페이지를 새로 reload하는 웹페이지라면 위의 방법을 써도 무방합니다. 하지만 현재 저의 웹 페이지는 SPA 형식으로 되어있습니다. 즉 다른 페이지를 넘어갈 때 jQuery의 Pjax를 이용하여 특정 컨텐츠만 변경하는 방식으로 구성 되어있습니다. 따라서 저는 위의 코드를 fetch 되는 페이지 안에다가 넣어주어야 했습니다. 그래야만 fetch 되는 페이지에서 MathJax까지 같이 불러와서 사용이 가능했기 때문입니다.

하지만 여기서 문제점을 발견했습니다. Pjax를 통해 fetch page를 하고 다시 수식이 있는 페이지로 fetch하여 돌아왔을 때 위의 MathJax의 script tag가 불러오지 않아 페이지의 수식이 깨지는 문제점이 있었습니다. 따라서 HTML body tag 안에 아래의 JavaScript 코드의 함수를 추가하여 해결했습니다. 

# JavaScript code
function initMathJax() {
    MathJax = null;
    if(typeof(MathJax) !== 'undefined') {
        MathJax.Hub.Queue(["Typeset", MathJax.Hub]);
    }
}

# HTML code
<body>
    <script src="//cdn.mathjax.org/mathjax/latest/MathJax.js?config=TeX-AMS-MML_HTMLorMML"></script>
    <script>
        initMathJax()
    </script>
</body>
