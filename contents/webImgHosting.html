GitHub Web Hosting 이미지 불러오기

작성자: 여행 초짜
작성일: 2022.08.23

이번 글에서는 구글 드라이브에 있는 이미지를 웹으로 불러와서 볼 수 있게 해주는 방법에 대해 알아보도록 하겠습니다.

저는 GitHub 웹 호스팅 기능을 통해 블로그를 운영하고 있습니다. 저는 인공지능 개발 글 뿐아니라 여행글도 같이 적고있는데, 여행 글을 적기 위해서는 많은 사진이 필요합니다. 그런데 다들 아시다시피 GitHub 웹 호스팅을 위한 용량은 1 GB로 제한을 두고 있습니다. 따라서 모든 사진을 GitHub에 올려놓고 사용하기에는 턱없이 부족한 용량이라 이러한 문제점을 해결해야 했었습니다. 그래서 사진을 구글 드라이브에 넣어놓고 URL을 통해 불러오는 방법을 알아보다가 찾은 해결 방법을 공유하도록 하겠습니다.


구글 드라이브 이미지 공유
먼저 구글 드라이브에 올린 이미지를 공유해야합니다. 이미지 우클릭 → 링크 생성에 들어갑니다.

그리고 액세스 범위를 "링크가 있는 모든 사용자"로 변경 합니다. 그런 후 생성된 링크를 복사합니다.

구글 드라이브 이미지 액세스 범위 수정


그리고 복사한 이미지 링크를 아래에 입력하고 convert link 버튼을 누르게 되면 이미지 창이 브라우저에 나타날 것입니다. 그 이미지 창의 URL 주소를 img 태그의 src로 넣어주면 됩니다. 현재 이 글에 있는 이미지도 모두 이러한 방법으로 불러온 것입니다.


그리고 링크 변경해주는 자바스크립트 코드는 아래에 첨부하도록 하겠습니다. 











function convertLink() {
    var obj = document.getElementById("googleDriveUrl")
    var name = obj.value;
    var st = 'file/d/';
    var tr = '/view';
    var prefix = 'https://lh3.google.com/u/0/d/';

    if (name === '') {
        alert('링크를 입력해주세요.');
    } else if (name.indexOf(st) === -1 && name.indexOf(tr) === -1) {
        alert('링크가 올바르지 않습니다.');
    } else {
        name = prefix + name.substring(name.indexOf(st)+st.length, name.indexOf(tr));
        window.open(name);
        obj.value = null;
    }
}
