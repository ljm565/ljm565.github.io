Knowledge-guided Context Optimization (KgCoOp)

작성자: 여행 초짜

시작하기 앞서 틀린 부분이 있을 수 있으니, 틀린 부분이 있다면 지적해주시면 감사하겠습니다.

이번에 소개할 논문은 Knowledge-guided Context Optimization (KgCoOp) 입니다. 이 모델은 기존 CoOp와 CoCoOp가 가지던 일반화 문제점을 해결하고 성능 향상을 위해 제안한 모델입니다.

CoOp와 CoCoOp는 CoOp 설명글과 CoCoOp 설명글을 참고하시기 바랍니다.

KgCoOp

“ KgCoOp의 동기
Motivation of CoCoOp ”

저자는 여전히 unseen class에 대해 CoOp, CoCoOp가 textual generalization이 안좋다는 문제점을 제기합니다.

  1. Hand-crafted prompt (CLIP처럼 자연어 prompt)와 learnable soft prompt의 embedding discrepancy를 줄임으로써 지식 망각을 방지할 수 있음.
  2. 위와 같이 hand-crafted prompt와 거리를 좁히도록 학습하는 loss를 추가함으로써, KgCoOp는 seen/unseen 클래스에 대해 모두 차별적인 prompt를 구성할 수 있음.

저자들은 자연어 prompt와 CoOp에서 처음 제안한 soft prompt와의 간극을 줄이는 loss를 추가함으로써 더 좋은 결과를 보여줄 수 있었다고 언급합니다.

“ KgCoOp의 구조 & 학습
KgCoOp Architecture & Traning ”

1. KgCoOp 구조
KgCoOp의 구조는 learnable prompt (soft prompt)를 받는 text encoder, 이미지를 받는 image encoder만 본다면 동일합니다. 다만 learnable prompt 뿐만 아니라 CLIP처럼 자연어 prompt를 받는 text encoder 하나가 더 추가되었습니다. CoOp의 구조는 CoOp 설명글을 참고하시기 바랍니다.

2. KgCoOp 학습
저자는 두 개의 text encoder의 결과를 아래와 같은 mean squared error loss를 사용하여 자연어 prompt와 soft prompt의 간극을 줄이는 방식을 contrastive learning에 덧붙여서 학습합니다. 저자는 zero-shot 일반화 성능 저하와 두 prompt 사이의 거리가 양의 상관관계에 있다고 주장하고, 따라서 이렇게 두 prompt의 간극을 줄이면서 학습하는 것이 unseen class에 대해 일반화를 더 잘하게 할 수 있게끔 해준다고 말합니다.

\[L_{kg}=\frac{1}{N_c}\sum^{N_c}_{i=1} \lVert w_i - w_i^{clip} \rVert_2^2\]


\(L_{ce}\)는 일반적인 cross-entropy loss와 동일하고 최종 loss는 아래와 같이 \(L_{kg}\)에 coefficient를 주어 계산합니다.

\[L_{ce} + \lambda L_{kg}\]

“ KgCoOp 결과
Results of KgCoOp ”

저자들은 4가지 모델에 대해서 seen class (학습 때 사용한 클래스), unseen class (zero-shot)에 대해 실험을 수행하여 성능을 비교합니다. 아래 결과 표에서 Base, New, H는 각각 seen class, unseen class, seen class + unseen class의 결과입니다. KgCoOp는 seen/unseen class에 대해 전반적으로 다른 모델들보다 좋은 결과를 보여줍니다.

그리고 각 데이터별로 학습한 결과를 비교하는데 여전히 zero-shot에 대해서는 CLIP이 좋은 성능을 보이지만, seen class까지 고려한다면 KgCoOp가 더 좋다고 주장합니다.
